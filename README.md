# 完形填空
=== 
> 单页面应用的Demo，在线的完形填空。

### 亮点   
* 动态判断解析选题和答案，丰富的交互细节
* 数据皆由解析JSON以及JS实现动态交互
* 动态判断对错、插入正确答案、拖拽、判分等实现


### 依赖
> JQuery 1.0+

### 兼容性  
> 未刻意考虑浏览器的兼容性，目前支持IE10+以及主流浏览器
